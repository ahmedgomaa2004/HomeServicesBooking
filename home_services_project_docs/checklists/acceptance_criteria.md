# Acceptance Criteria

## Public Website

- [ ] Home Page displays services.
- [ ] Service Details Page displays service information.
- [ ] Booking button opens Order Form with selected service.
- [ ] Order Form validates all required fields.
- [ ] Valid order is saved in SQL Server.
- [ ] Success message appears after saving.
- [ ] Error message appears if saving fails.

## Admin Login

- [ ] Login page exists.
- [ ] Email validation works.
- [ ] Password minimum length validation works.
- [ ] Invalid credentials show `بيانات غير صحيحة`.
- [ ] Valid credentials redirect to Dashboard.
- [ ] Logout works.
- [ ] Admin pages are protected.

## Dashboard

- [ ] Total orders count is correct.
- [ ] Pending orders count is correct.
- [ ] Done orders count is correct.
- [ ] Last 10 orders table is displayed.

## Orders Management

- [ ] Orders table displays required columns.
- [ ] Search by name works.
- [ ] Search by phone works.
- [ ] Filter by status works.
- [ ] Pagination displays 10 rows per page.
- [ ] Details button works.
- [ ] Status update works.

## Order Details

- [ ] All order details are displayed.
- [ ] Status dropdown contains Pending, In Progress, Done.
- [ ] Save button updates database.
- [ ] Success message appears.

## Services Management

- [ ] Services list works.
- [ ] Create service works.
- [ ] Edit service works.
- [ ] Delete or deactivate service works.
- [ ] Validation works.

## Delivery

- [ ] Source code ready.
- [ ] SQL script ready.
- [ ] README ready.
- [ ] Screenshots ready.
- [ ] Demo video ready.
