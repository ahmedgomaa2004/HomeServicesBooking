# Delivery Checklist

## Source Code

- [ ] المشروع كامل داخل Folder واضح.
- [ ] لا توجد ملفات bin/obj غير ضرورية في النسخة النهائية.
- [ ] لا توجد أسرار أو Passwords حساسة داخل الكود.
- [ ] Connection String موضحة في README ويمكن تعديلها.

## Database

- [ ] Migration موجودة.
- [ ] SQL Script جاهز.
- [ ] Seed Data موثق.
- [ ] Admin الافتراضي موضح أو قابل للتغيير.

## README

README النهائي يجب أن يحتوي على:

- وصف المشروع.
- المتطلبات اللازمة للتشغيل.
- خطوات تشغيل المشروع.
- خطوات إنشاء قاعدة البيانات.
- بيانات الأدمن الافتراضية.
- شرح سريع للصفحات.
- ملاحظات مهمة.

## Demo Video

مدة الفيديو: 2-3 دقائق.

4. تسجيل دخول الأدمن.
5. عرض Dashboard.
6. فتح الطلبات.
7. البحث أو الفلترة.
8. فتح الطلب الجديد.

10. عرض إدارة الخدمات.

## Final Acceptance

- [ ] كل متطلبات العميلة منفذة.
- [ ] لا توجد أخطاء واضحة.
- [ ] المشروع يعمل من أول تشغيل حسب README.
- [ ] تم توضيح أي بيانات افتراضية.
- [ ] تم تسليم الملفات المطلوبة.
