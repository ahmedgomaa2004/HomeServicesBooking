# Home Services Booking - Documentation Pack

هذا الباكدج يحتوي على ملفات التخطيط والتنفيذ التي سنبدأ منها مشروع ASP.NET Core MVC + SQL Server.

## أهم الملفات

- `START_HERE.md` نقطة البداية.
- `plan.md` الخطة الكاملة.
- `01_requirements.md` المتطلبات.
- `02_architecture.md` المعمارية.
- `03_database_design.md` تصميم قاعدة البيانات.
- `TASK_BOARD.md` لوحة المهام.
- `phases/` ملفات تنفيذ المراحل بالتفصيل.
- `agent-prompts/` برومبتات جاهزة لـ Windsurf و Codex.
- `10_client_message_price_change.md` رسالة تغيير السعر بسبب تغير النطاق.

## طريقة الاستخدام

1. افتح `START_HERE.md`.
2. اقرأ `plan.md`.
3. شغل Windsurf بالبرومبت الموجود في `agent-prompts/windsurf_main_prompt.md`.
4. نفذ المراحل واحدة واحدة من مجلد `phases/`.
5. استخدم `TASK_BOARD.md` لتتبع التقدم.
6. استخدم `checklists/acceptance_criteria.md` في نهاية المشروع.

## ملاحظة مهمة

لا تبدأ تنفيذ كامل قبل توثيق تغيير السعر مع العميلة، لأن المشروع تحول من تعديل مشروع موجود إلى بناء مشروع كامل من الصفر.
